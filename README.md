## 🧠 Project Overview

This project predicts **Data Science salaries** based on multiple features like:

- Company Name  
- Job Role  
- Location  
- Employment Status  
- Ratings  

👉 Uses **Random Forest Regressor** to output **real salary values (not categories)**.

---

## ⚙️ Features

✨ Real-time salary prediction  
✨ Clean and interactive UI  
✨ Regression-based ML model  
✨ Handles categorical data  
✨ Fast and lightweight deployment  

---

## 📊 Model Details

- Algorithm: **RandomForestRegressor**
- Input: Structured job-related features  
- Output: **Continuous salary value (₹ / $)**
